Hughes Meyer Studio

Website for a London based art studio. This is an ongoing project, a multi page website with a very strict space handling design. I have used Javascript and jQuery extensively for creating some of the features of the site, like a space optimizing responsive grid for navigating between elements inside a large project, a lens zoom effect to reveal subtle details and semi-fixed elements to accommodate the design. As the site has many pages using the same layout, I utilized the features of Gulp and Swig for templating.

Map customization tool

A website that enables users to customize colors, visibility and line thickness of the areas on their maps using a graphic interface, and then generates the code for them.

A simple task, but it led me to discover many great features like the shutter out vertical effect used on the menu items, 3D CSS transitions used on the visibility radio buttons, a great styler for number inputs and of course the customization of Google maps.

Sarbat

A single page website, created based on a free Behance web design by Vishnu Sathyan. I have used Sass to style the page. The gallery section presented a great opportunity for trying out Masonry, a grid layout library and experiment with CSS transitions for the captions. The conical gradient skills progression bar was drawn using canvas. In order to provide a great user experience across all devices I have made the site responsive.

Kaleidoscope 1

While completing an edX course on HTML 5 by W3C I came across the canvas element and got very enthusiastic really quickly. This is one of the experiments that I have built using Javascript on codepen. It is a kaleidoscope like effect with hexagons mirroring moving dots inside them.

Kaleidoscope 2

The second piece of my ongoing Kaleidoscope series, this time I used the sides of triangles to mirror the movement of circular lines inside.

Tetris Xmas

A holiday themed project to practice CSS animations and to explore more features of SASS. Creating the tetris snowflakes using the "box-shadow" property with multiple arguments was inspired by a codepen I have stumbled upon recently. Studying the code of my fellow codepeners is always a great way to discover exciting new features.

Spider Club

This project was an enjoyable exercise to practice bootstrap. I have mostly experimented with the responsive grid layout provided by the framework, it was a great learning experience to solve the problem of the need for a margin versus bootstrap's margin unfriendly column layout with grid inception. The project also presented a valuable lesson about optimizing taxing tasks in Javascript with the debounce function.

Soap Bubbles

My first project using the canvas element, I had a great time experimenting with the math of the trajectories to bring the movement of the bubbles as close to reality as possible. Resembling the fact, that one can never encounter the exact same result when blowing real soap bubbles, I invoked random numbers to set the starting position and to some extent the initial radii of the bubbles, therefore every viewer has a unique experience.
